Program ini bertujuan untuk mengirimkan pembacaan rfid ke program python menggunakan serial dan melakukan pembacaan plat.
1. Pertama-tama upload file program arduino
2. Install yolov5-master pada github, install pada satu direktori dengan deploy.py
3. Jalakan program deploy.py pada python
4. Ganti best.pt yang berada di satu direktori dengan deploy.py dengan best.pt yang sudah kalian training jika ingin mengunakan data pelatihan kalian
5. Sesuaikan alamat kamera dengan kamera yang sesuai coba ganti mengunakan angka (1,2,3,4,5) sampai kamera konek
6. Selamat Mencoba

Penggunaan Program
1. tempelkan kartu mifare pada rfid 
2. hadapkan plat pada kamera karena ketika rfid terbaca maka kamera akan langsung melakukan penangkapan gambar
3. setelah itu gambar akan otomatis tersimpan di directory baru
